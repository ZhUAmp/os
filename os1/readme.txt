myshell — простая оболочка командной строки для UNIX

Использование:
--------------
Запуск:
  ./myshell         — запускает в интерактивном режиме
  ./myshell файл    — выполняет команды из указанного файла

Поддерживаемые команды:
-----------------------
cd <dir>       — сменить каталог (если не указано, покажет текущий)
clr            — очистить экран
dir <dir>      — вывести содержимое каталога
environ        — вывести все переменные среды
echo <text>    — вывести текст
help           — вывести это руководство
pause          — приостановить выполнение до нажатия Enter
quit           — завершить работу оболочки

Дополнительно:
--------------
>   — перенаправление вывода в файл (перезапись)
>>  — перенаправление с дозаписью
<   — (не реализовано, можно расширить)
&   — выполнение команды в фоновом режиме

Пример:
-------
echo Hello World > file.txt
dir / >> dirlog.txt
./myshell script.txt
sleep 10 &

О переменных среды:
-------------------
Переменная shell содержит путь к исполняемому файлу myshell.
Переменная parent содержит путь к myshell для дочерних процессов.

Эта оболочка реализована на C и предназначена для базового ознакомления с концепциями командных интерпретаторов.
